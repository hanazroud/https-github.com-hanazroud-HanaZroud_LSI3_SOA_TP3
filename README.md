# https-github.com-hanazroud-HanaZroud_LSI3_SOA_TP3
Calculatrice REST

Description >>>>

Ce projet est un service REST implémenté en Java avec JAX-RS (Jersey).
Il propose une API pour effectuer des opérations de calcul : addition, soustraction, multiplication et division.

Le projet est packagé avec Maven et déployé sur Tomcat.

Structure du projet  >>>>>


Calculatrice-rest/
│
├─ src/main/java/com/example/calculatrice/  # Classes Java
│   ├─ Calculatrice.java
│   ├─ CalculRest.java
│   ├─ Operation.java
│   └─ RestApplication.java
├─ src/main/webapp/WEB-INF/web.xml          # Configuration web
├─ pom.xml                                  # Configuration Maven
└─ target/     

Environnement technique utilisé ✅ (Java, Maven, Tomcat, Jersey)
